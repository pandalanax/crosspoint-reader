# Contacts

Offline contact book. Reads a pre-generated JSON file from the SD card and displays an alphabetical list with letter-based quick navigation.

## Setup

1. Export your contacts as a `.vcf` (vCard) file
2. Run the converter script on your computer:
   ```
   python3 scripts/vcf_to_contacts_json.py contacts.vcf contacts.json
   ```
3. Copy `contacts.json` to `/.crosspoint/contacts.json` on the SD card

The script extracts names, phone numbers, emails, and organization. Photos and other fields are ignored. The output is sorted alphabetically and typically under 100 KB even for hundreds of contacts.

## How It Works

- Purely offline -- no WiFi, no sync, no server
- Reads `/.crosspoint/contacts.json` on open
- To update contacts, re-run the script and copy the new file to the SD card

## Controls

### List View

| Button | Action |
|--------|--------|
| Up / Down | Scroll one contact at a time |
| PageForward | Jump to next letter (A -> B -> C...) |
| PageBack | Jump to previous letter |
| Confirm | Open contact detail |
| Back | Go home |

### Detail View

| Button | Action |
|--------|--------|
| Back | Return to list (position preserved) |

## Files

| Path | Purpose |
|------|---------|
| `/.crosspoint/contacts.json` | Contact data (generated by script, copied manually) |
| `scripts/vcf_to_contacts_json.py` | Converter script (runs on PC, not on device) |

## JSON Format

The script outputs a compact JSON array:
```json
[
  {"n": "Alice Smith", "p": ["+31612345678"], "e": ["alice@example.com"], "o": "Acme Inc"},
  {"n": "Bob Jones", "p": ["+44712345678", "+44207654321"]}
]
```

Keys: `n` = name, `p` = phones, `e` = emails, `o` = organization. All optional except `n`.
